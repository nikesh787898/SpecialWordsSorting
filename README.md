# SpecialWordsSorting

Given a list/array of names(String) sort them such that each name is followed by a name which starts with the last character of the previous name. 
# input 
[ 
Luis 
Hector 
Selena 
Emmanuel 
Amish 
] 

# output: 
[ 
Emmanuel 
Luis 
Selena 
Amish 
Hector 
]
